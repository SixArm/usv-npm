# USV NPM: Unicode Separated Values (USV) with Node Package Manager (NPM)

Initialize:

```sh
➜ mkdir usv-npm && cd $_
➜ git init
➜ git remote add origin git@github.com:SixArm/usv-npm.git
➜ git branch -M main
➜ git push -u origin main
```

Add NPM:

```sh
➜ npm init
➜ npm --version
10.5.2
```

Add TypeScript:

```sh
➜ npm install -g typescript
➜ tsc -v
Version 5.4.5
```
